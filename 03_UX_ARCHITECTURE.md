# 03. UX ARCHITECTURE — IDENTITY PRISM ENGINE (IPE)
**Status:** Locked (v6.0) — Neuro-Map & Socratic Interaction
**Design Paradigm:** Editorial Manuscript & Logic Traceability

---

## 🎨 1. GLOBAL DESIGN SYSTEM (The "Vibe" Engine)
The interface uses a **State-Based Style Compiler** that reads environmental metadata to alter the DOM.

- **The "Deep Logic" State:** (Triggered by focus/techno/night)
  - Typography: Strict Monospace (JetBrains Mono) for logic, high-contrast serif for narrative.
  - Layout: Tight grid gaps, dense text blocks, absolute minimalism.
  - Color: Pure monochrome (Black/White/Gray).
- **The "Expansive" State:** (Triggered by creation/day)
  - Typography: Clean humanist Sans-Serif (Inter).
  - Layout: Looser line-heights, wider padding.
  - Visibility: Audio "Strategic Sparks" elevated visually.

---

## 🌓 2. THE ASYMMETRIC DUALITY

### 🔷 ENVIRONMENT A: The Neuro-Map (Public Proxy)
- **Vibe:** Editorial Manuscript. Linear scrolling. Max width 600px.
- **The 6-Layer Scroll:**
  1. **Identity:** Hero text + Live State Capsule.
  2. **Intelligence:** Text blocks with [Trace Logic] neural lines.
  3. **Scope:** Capability Range axis [ UX —— Product —— SYSTEMS —— Ops ].
  4. **Proof:** 3-Step Reveal accordions (Surface → Logic → System).
  5. **Output:** Terminal-style feed of sparks and commits.
  6. **Trajectory:** Bold Directional Vector statement.

### 🔷 ENVIRONMENT B: The Dialogue Hub (Owner HUD)
- **Vibe:** Socratic Command Center. Conversational interface.
- **Views:**
  - **The Socratic Inbox:** Chat-based dissonance resolution with the Brain.
  - **The HOLD Triage:** Swipe-friendly mobile list for raw telemetry.
  - **The Temptation Analyzer:** Split-screen ROI vs. Spirit Decay gauge.

### 🔷 ENVIRONMENT C: The Complexity Port (Evaluator Intake)
- **Vibe:** Silent Metabolism. Absolute elimination of friction.
- **Views:**
  - **The Silent Drop-Zone:** Massive text input for JDs/problem statements.
  - **The Simulated Audit:** Structured report generated instantly (Diagnosis → Proof → Gate).

---

## 🧩 3. GLOBAL INTERACTION RULES
- **No Modals:** All deep-dives happen via inline expansion to preserve context.
- **No Placeholders:** Layers without data collapse; no "Coming Soon" states.
- **SVG Neural Threads:** Visual lines connecting abstract claims to physical proof.
